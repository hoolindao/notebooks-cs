# Python变量赋值发生了什么？

```python
a = 'ABC'
b = a
a = 'XYZ'
print(b)

```

```text
step1
| 变量名 | 指向 | 数据空间   |
|-------------------------|
|  a    | ->  | str1 'ABC'|

step2
| 变量名 | 指向 | 数据空间   |
|-------------------------|
|  a    | ->  | str1 'ABC'|
|  b    | ->  | str2 'ABC'|

step3
| 变量名 | 指向 | 数据空间   |
|-------------------------|
|       |     | str1 'ABC'|
|  b    | ->  | str2 'ABC'|
|  a    | ->  | str3 'XYZ'|
```
