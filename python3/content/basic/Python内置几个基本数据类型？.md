#### 字符串 例子
```python
'word'

"I'm OK"

'I\'m OK'
    
'I say: \"very goo\"'

r'\\\\\t\\\\\'

'''line1
line2
line3'''
```
##### 字符串小结
- `''`与`""`只是标识符，标明字符串类型，其本身不是字符串的一部分
  - 如`print('word')`输出内容为 word 
- 当字符串内容含有标识符时，需用转义符`\`告诉Python，否则将不能实现预期效果
  - 如`print('I\'m OK')`能真确执行并输出和`print('I'm OK')`会报错
  - `\t`制表符 `\n`换行符
- 与转义相关的还有`r''`表示引号内部的字符串默认不转义
- `''' '''`多行字符串
